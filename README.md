Поддерживаемые игры:

CS: Source (OrangeBox), CS: Source (v34), CS: GO

Описание:

Записывает время, никнейм, steamid и ip игрока в файл.
Будет полезно если хотите знать сколько игроков побывало задень на сервере.

1) Плагин создает каждый день новый файл.

1.1) Имя файла это день, месяц, год.

2) Записывает игроков один раз в день.

Требования:

Sourcemod (1.11.0.6916) (Для smx файл, sp файл можете на своей версии скомпилировать)

Установка:

Раскидать файлы по папкам.
